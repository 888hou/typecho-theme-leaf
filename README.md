## 主题介绍

我移植的[第一款主题：BOOK](http://www.ihewro.com/archives/367/) 是为我写日记的一个博客准备的，而这款也是同一个作者写的，相比简洁，更大气一点！这款最近我准备完善一点自己换上。所以握有移植到typecho上面了。

如果有什么建议可以提。

## 主题特色

没什么特色的，就是简洁，没有任何多余的页面，只有一个归档、关于我、首页三个页面。以及错误404页面。
